Make a game with firebase integrated

Created a 2D idle clicker game to collect data and store into firebase.

About this Project

I create a 2D clicker game with firebase integrated.

Work Process

Planning

I researched on how a 2D game works before I started as I had no experience in creating a 2D game. Then I created in game assets needed to be used in my game with Pixel Studio as I wanted a simple pixel artstyle.



Then I created a firebase connected to my unity game so that it can collect user data and provide statistics.



Designing

I need user data so I created a sign in and sign up page so I can get the user details and store each user's data during the game.

Sign In & Sign Up



Once they sign up for a new account (or sign in), their registered data will reflect into firebase.



As the player / user play the game, their score will be sent to the database.



Results & Takeaways

With firebase database integrated into my game, it provided me with a valuable learning experience. By incorporating Firebase's capabilities, such as real-time database updates optimized the game's performance and enhanced user engagement. This integration streamlined data management processes, enabling efficient tracking of player progress and in-game achievements. Looking back, the project illuminated the versatility of Firebase in enhancing gameplay dynamics and fostering interactive experiences, while offering valuable lessons for future endeavors in leveraging cloud-based solutions for game development.
