3D Ice Cream model van referenced from Alexandr Krainuk

This is my first project assignment on 3D art. The reference image is from Alexandr Krainuk.

About this Project

Build a 3D model based on the illustration given, which is an ice cream truck illustrated by Alexandr Krainuk.

Work Process

Planning

Since the image is 2D, I will have to analyse the concept image and replicate the volumetric shapes in 3D space.

I will also have to decide on the textures to be applied on the ice cream van so that it will fit the ice cream and look alike to the image.

The image below shows the analyse on the 2D illustration. Where I differentiate the the shapes with different colors to help me see and visualise the model better.

Designing

After I had analysed the illustration, I will have to start building the base of the van. Mainly used extrude function in creating the base from a cube object. Below shows the base process of the project:

The van itself is in low poly. However, I had made a high poly on the wheels and below is how it looks like in Adobe Substance Painter.

The van will also have different textures, so I had to choose the best matching materials suited to the van in substance painter. Below is how it I did my textures.

Results & Takeaways

The 2D illustration had turned into a 3D model.

Through this project, I am familiarized with the various 3D tools and workflows in the 3D modelling, UV unwrap, texturing, lighting, real-time & pre-rendered presentation.

I had also learnt to analyse references before getting to work on the model from scratch.
